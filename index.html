MCM LTER Bibliography

This searchable database includes all MCM-supported publications and key foundational literature. Results may take a moment to load. PDFs are available for most publications, which can be viewed or downloaded by anyone by who joins the LTER-MCM Zotero group. Here's how:

  1. Create a free Zotero account at zotero.org (click Register at the top right).
  2. Log in and visit the LTER-MCM Zotero group.
  3. Click Join (button on the right side).
  4. Once approved, you'll have access to PDFs via the group library.

To search, enter a term below (e.g., title or author). You can also filter by item type and change the sort order. Please note that this search tool uses the Zotero API and supports only basic citation fields; it does not search abstracts, keyword tags, or full text. To browse by keyword, use the Zotero group library and select tags from the sidebar.

For more advanced features, such as full-text search (if enabled) and integration with word processors for citation management, we recommend using the Zotero desktop app. Refer to the Zotero documentation for further details.

Questions or corrections? Contact the MCM LTER Information Management Team at im@mcmlter.org.
